Using the ADA Issue Documentation Log

      
Documentation protects managers from vague decision-making. When an ADA parking, accessible route, signage, striping, slope, drainage, or pavement issue is observed, the record should show when it was noticed, where it was located, what photos were taken, what follow-up was recommended, and whether it needs measurement or professional review.

      
Use this log before paving, sealcoat, striping, concrete repair, board approval, budget planning, insurance review, or contractor scope approval. Save the output with dated photos, proposals, field measurements, emails, and board notes so the decision trail is clear later.
